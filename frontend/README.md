# Renato Vieira Portfolio Frontend

Portfólio pessoal e área administrativa em React, Vite, Tailwind CSS, React Router, Axios e Framer Motion.

## Instalação

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

Configure `VITE_API_URL` com a URL da API. Se frontend e backend estiverem no mesmo domínio, use:

```env
VITE_API_URL=/api
```

Se o backend estiver em outro domínio, use a URL pública dele, por exemplo `https://api.seu-dominio.com/api`.

## Build

```bash
npm run build
```
